  <name>Task 1: videoTaskStore + VideoParamsForm + VideoPlayer 组件</name>
  <files>src/renderer/stores/videoTaskStore.ts, src/renderer/components/VideoParamsForm.vue, src/renderer/components/VideoPlayer.vue</files>
  <action>
    1. 创建 src/renderer/stores/videoTaskStore.ts（Pinia）:
       - state: { tasks: VideoTask[], submitting: boolean }
       - submitTask(params: VideoSubmitParams): 调用 video:submit IPC，将返回的 task 加入列表，
         启动轮询（setInterval 3000ms 调用 video:poll，status 为 succeed/failed 时 clearInterval）
       - 轮询时更新对应 task 的 status、video_url、cover_url

    2. 创建 src/renderer/components/VideoParamsForm.vue:
       - 时长：IonRange，min=3 max=15 step=1，显示当前值（单位：秒）
       - 宽高比：3 个图标按钮组（16:9 / 9:16 / 1:1），选中态高亮
       - 模式：IonSegment，std="标准" pro="专业" multi_shot="多镜头"
         multi_shot 选项旁加 IonBadge 显示"多镜头"标识，颜色区分于 std/pro
       - 首帧/尾帧：各一个上传按钮，选择后显示 64x64 缩略图预览，支持清除
       - emit: update:modelValue（VideoSubmitParams 对象）

    3. 创建 src/renderer/components/VideoPlayer.vue:
       - props: { src: string }（视频 URL）
       - 使用原生 <video> 元素，loop autoplay muted（静音自动播放）
       - 显示进度条（input[type=range] 绑定 currentTime/duration）
       - 暂停/播放切换按钮
       - 样式：圆角卡片内嵌，宽度 100%，高度自适应
  </action>
